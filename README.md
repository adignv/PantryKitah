# PantriKita 🛒

**PantriKita** adalah aplikasi Android berbasis *Universal Design* yang dirancang untuk membantu masyarakat umum mengelola persediaan bahan makanan di dapur dan mencegah *food waste* (sampah makanan). Aplikasi ini secara khusus dilengkapi dengan fitur aksesibilitas tingkat tinggi, sehingga ramah dan sangat mudah digunakan oleh penyandang disabilitas, khususnya disabilitas netra dan *low vision*.

---

## ✨ Fitur Utama

* 📷 **Pemindai Barcode Instan (Vision API):** Menambahkan data barang secara otomatis dan cepat hanya dengan memindai *barcode* kemasan produk menggunakan kamera, tanpa perlu mengetik manual.
* 🔊 **Text-to-Speech (Suara Aksesibilitas):** Aplikasi akan membacakan nama bahan makanan beserta tanggal kedaluwarsanya secara lisan saat item diketuk. Sangat krusial bagi pengguna tunanetra.
* 📋 **Manajemen Stok Visual:** Pencatatan rapi dengan antarmuka yang bersih untuk memantau daftar barang dan sisa waktu sebelum kedaluwarsa.
* 🎨 **Desain UI Modern (Glassmorphism):** Tampilan cerah bergaya iOS dengan kontras tinggi dan teks tebal (*bold*) yang mempermudah navigasi bagi pengguna dengan penglihatan rendah (*low vision*).

## 🛠️ Teknologi yang Digunakan

* **Bahasa Pemrograman:** Kotlin
* **Lingkungan Pengembangan:** Android Studio
* **Machine Learning:** Google Play Services Code Scanner (Google ML Kit)
* **Aksesibilitas:** Android Native TextToSpeech (TTS) Engine
* **UI/UX:** XML Material Design Components
